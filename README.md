# Проверка сенсорных способностей

Приложение для тестирования сенсорных способностей человека.

## Функциональность
- Тест скорости реакции
- Тест цветовосприятия  
- Тест слуха

## Установка и запуск
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python main.py`
